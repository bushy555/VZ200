Disk images of three graphical demos, which will run under the emulator WinVZ, written by Guy Thomason.
(   WinVZ available at http://intertek00.customer.netspace.net.au/vz200/   )
15/Feb/2009


Juggler ][
----------
Run WinVZ
Mount jug1.dsk to drive 1
Mount jug2.dsk to drive 2
DRIVE2 
RUN"JUGGLER2"

Type <1> to lower the top of memory
Press <Enter> to run the demo

- Sit back and watch -



Panic/Wormhole
--------------
Run WinVZ
Mount panic1.dsk to drive 1
Mount panic2.dsk to drive 2
RUN"PANIC"
Type <1> to lower the top of memory
Press <Enter> to run the demo
When asked, mount worm1.dsk to drive 2

- Sit back and watch -
